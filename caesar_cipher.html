Como funciona:

A Cifra de César é um dos métodos de criptografia mais antigos, usado por Júlio César para proteger mensagens militares. Ela funciona substituindo cada letra do alfabeto por outra que está um número fixo de posições (deslocamento) à frente.

Exemplo com deslocamento = 3:
• A → D, B → E, C → F, ..., Z → C (o alfabeto "dá a volta").
• "OLA" se torna "ROD".

Para decifrar, basta retroceder o mesmo número de posições. Experimente ajustar as rodas abaixo para ver a transformação em tempo real!
